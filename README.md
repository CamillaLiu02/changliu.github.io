# Chang Liu — Portfolio

Live site: https://changliu.github.io

A personal portfolio website showcasing my work in robotics & HCI, full-stack development, and UX/UI.

---

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- MDX (project case studies)

---

## Project Structure
- `app/` — pages (Home, Projects, About, Resume, Contact)
- `content/projects/` — MDX project writeups
- `public/` — images, resume PDF, static assets
